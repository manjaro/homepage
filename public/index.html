Grows with you

Built a powerful Arch-Linux foundation, with speed, efficiency and cutting edge software, Manjaro makes it stable and takes full advantage of your computer hardware. It is designed to make beginners happy and grow with them while they are becoming advanced users. It is easy to install even for beginners and comes with all the apps you need every day.

Features that will help you

Manjaro come preinstalled with GIMP, which is a very powerful image manipulation program. For drawing Krita will fulfill most of your needs, while Darktable brings a top-notch and free photography suite. For simple or advanced video-editing, Kdenlive and OpenShot will do almost anything you need. There are also professional video-editing tools like Lightworks. 3D-Modelling can be done using Blender. Most of this software is completely free available and can be easily and secure installed by using Manjaro’s integrated package manager.

You can also use Torrents to download Manjaro. Just download the *.torrent file for the 64 bit or 32 bit version and open it with a torrent program like qBittorrent.

Be secure

We encourage you to check if the download was not modified by some malicious person or by an error while downloading. As a simple check, you can generate the SHA1-sum of the downloaded file and compare it with the one we provide you here. Here you can find information how to generate such a sum.

To even further enhance the security, verify the downloaded file with a GPG (GNU Privacy Guard) signature that will ensure that the file was not modified and was actually created by the Manjaro developers. Here you can find information how to use it.

Getting started

After downloading and checking the file, you can put it on a DVD or a USB drive. You will find step-by-step guides in the Manjaro Wiki.
